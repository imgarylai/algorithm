# 27. Remove Element

[27. Remove Element](https://leetcode.com/problems/remove-element/)

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        i = 0
        j = len(nums) - 1
        
        while i < j:
            if nums[i] == val:
                if nums[j] == val:
                    j -= 1
                else:
                    nums[i], nums[j] = nums[j], nums[i]
                    i += 1
                    j -= 1
            else:
                i += 1
        
        k = sum([1 for num in nums if num != val])
        return k
```
